Software Requirements Specification

for devsInc

Version 1.0
Prepared by Jian Zhou
IOD student
18 Feb 2023

devsInc want to support people with playing tabletop games from online and they require a Dice Generator. 

Functionalities
1. The ability to change the number of dice faces 
2. A button to get a random number accorrding to the input number of dice faces
3. A roll history listed onto a table
4. Try to use a responsive page content to adapt different screen sizes

Basic Flow:
1. User lands on home page
2. User types in number of faces for the dice
3. User presses "Roll button"
4. User gets result and the history table updates
